# Vue Project Application
A responsive one-page application developed with Vue 3, Sass, and Bootstrap 5 as part of a front-end job application.

## Features
- VueJS3
- HTML 5
- CSS 3 
- Mobile-first design
- Bootstrap 5 layout
- Sass custom styling

## Project Setup
- Install Node.js (https://nodejs.org/pt)
  
- npm init vue@latest
- npm install
- npm run dev
